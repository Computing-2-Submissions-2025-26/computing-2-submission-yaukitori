get_feedback(secret, guess) → {Feedback}

Description:

  • Score a single guess against the secret code.

    A green peg means the note is correct and in the right position. A red peg means the note appears in the secret but in a different position. Each note in the secret can only account for one peg.

Source:
  • Mastermind.js, line 44
Example
get_feedback(["C","D","E","F"], ["C","E","D","F"])
// => { greens: 2, reds: 2 }
Parameters:
Name Type Description
secret Secret

The hidden code chosen by the computer.
guess Guess

The player's attempt, same length as secret.
Returns:

Object with greens and reds counts.
